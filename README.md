# Mô tả
Phiên bản từ điển 3 viết bằng MERN lượng từ trên server hơn 2800 từ

# Chạy server

Thực hiện các lệnh

```
cd server
npm install
npm start
```

# Chạy client 

```
cd client
npm install
npm start
```


## Backend

1. Sử dụng NodeJS, ExpressJS để viết server
2. MongoDB làm database
3. Mongoose làm ODM cho mongoDB

## Frontend

1. ReactJS để viết frontedn
2. Reactstrap để viết CSS [https://reactstrap.github.io/]
3. Axios để fetch data
4. React Router v6 để chuyển trang
5. React-chartjs để dựng đồ thị [https://github.com/reactchartjs/react-chartjs-2]
